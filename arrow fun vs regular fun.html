Difference between regular fun() and arrow fun()

'this' with Arrow function

  • Arrow function is introduced in ES6 2015
  • There is no binding of this with the arrow function.
  • Always represent global object (window)
  • Ex - If arrow function call on button then it still represent global object [object, window] like that

'this' Regular function

  • Regular function always there in js
  • In a regular function this keyword represents the object that called the function.
  • Ex - If you call the button, it represents the button. like [object,HTMLButtonElement] If you call a global object then represent a window object etc.
